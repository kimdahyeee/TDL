### 입력스트림 & 출력 스트림

---

스트림 크래스의 종류는 크게 **바이트 기반 스트림**과 **문자 기반 스트림**으로 나누어진다.

- **바이트** 기반 스트림
  - 입력 스트림
    - 최상위 클래스 : InputStream
    - 하위 클래스 (예) : XXXX**InputStream**
  - 출력 스트림
    - 최상위 클래스 : OutputStream
    - 하위 클래스 (예) : XXXX**OutputStream**

- **문자** 기반 스트림
  - 입력 스트림
    - 최상위 클래스 : Reader
    - 하위 클래스 (예) : XXXX**Reader**
  - 출력 스트림
    - 최상위 클래스 : Writer
    - 하위 클래스 (예) : XXXX**Writer**
